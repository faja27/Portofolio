## 🛠️ Built With

- **HTML5** — Semantic structure
- **CSS3** — Custom properties, grid, animations
- **Vanilla JavaScript** — No frameworks
- **Framer-free animations** — CSS keyframes & transitions
- **i18n system** — Custom bilingual switcher (EN / ID)

---

## ✨ Features

- 🌗 Dark theme with cyan & purple accents
- 🌐 Bilingual support — English & Indonesian (persists across pages)
- 📱 Fully responsive — mobile, tablet, desktop
- 🖼️ Certificate viewer — click card to view certificate image
- ⚡ Smooth scroll animations
- 🔗 Active nav highlighting per page
